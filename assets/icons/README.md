# Weather Icons

This directory should contain 48x48 pixel PNG weather icons.

## Required Icons

Create the following icon files (48x48 PNG format):

| Icon File | Description |
|-----------|-------------|
| sunny.png | Clear/sunny weather |
| partly_cloudy.png | Partly cloudy |
| cloudy.png | Cloudy/overcast |
| rain.png | Rain |
| rain_light.png | Light rain/drizzle |
| rain_heavy.png | Heavy rain |
| thunderstorm.png | Thunderstorms |
| snow.png | Snow |
| snow_light.png | Light snow |
| snow_heavy.png | Heavy snow/blizzard |
| sleet.png | Sleet |
| freezing_rain.png | Freezing rain |
| ice.png | Ice conditions |
| fog.png | Fog/mist/haze |
| windy.png | Windy/breezy |
| unknown.png | Default for unknown conditions |

## Sources

You can download open-source weather icons from:
- Material Design Icons: https://materialdesignicons.com/
- Weather Icons: https://erikflowers.github.io/weather-icons/
- OpenWeatherMap icons: https://openweathermap.org/weather-conditions

## Notes

- Icons should be 48x48 pixels for optimal display
- Use simple, high-contrast designs
- Icons will be scaled if needed but best to provide at exact size
